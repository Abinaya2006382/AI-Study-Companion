AI Study Companion
An intelligent web-based learning assistant that generates notes, quizzes, and tracks performance using AI.
🚀 Overview
AI Study Companion is designed to make learning smarter and more interactive. It provides:
Personalized AI-generated study notes
Interactive quizzes with instant correct answers
Performance tracking for students
✨ Features
📚 AI-generated study notes
❓ Auto-generated MCQs with instant feedback
📊 Score calculation and progress tracking
💾 Database integration for user data
⚡ Simple and lightweight Python + MySQL backend
🛠️ Tech Stack
Backend: Python (Flask) – app.py
Database Config: db_config.py & SQL script
Configuration: config.toml
Database: MySQL
